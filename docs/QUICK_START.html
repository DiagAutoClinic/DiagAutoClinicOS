DiagAutoClinicOS - AutoDiag Quick Start Guide

Welcome to AutoDiag Suite v0.0.1 (Alpha)

This guide will help you get started with the AutoDiag diagnostic dashboard.

Initial Setup

  1. Connect your diagnostic hardware (OBDLink MX+, GoDiag, etc.)
  2. Launch AutoDiag from the desktop shortcut
  3. Select your vehicle make and model
  4. Configure connection settings

Main Features

  • Dashboard Tab: System overview and quick actions
  • Diagnostics Tab: Vehicle system diagnostics
  • Live Data Tab: Real-time parameter monitoring
  • Special Functions Tab: Advanced vehicle operations
  • Calibrations Tab: Reset and calibration procedures
  • Advanced Tab: Technical tools
  • Security Tab: Access control

Hardware Compatibility

  • OBDLink MX+ / EX
  • GoDiag GT100 / GT100 Plus GPT
  • Scanmatik 2
  • Generic ELM327 interfaces

Focus Brands: Ford and GM vehicles (enhanced support)
