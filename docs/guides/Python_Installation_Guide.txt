AUTO DIAG PRO - PYTHON INSTALLATION GUIDE
=========================================

This guide will help you install Python, which is required to run AutoDiag Pro.

STEP 1: Download Python
-----------------------
1. Open your web browser
2. Go to: https://www.python.org/downloads/
3. Click "Download Python 3.x.x" (latest version)
4. Run the downloaded installer

STEP 2: Install Python
----------------------
1. Run the Python installer you downloaded
2. IMPORTANT: Check the box "Add Python to PATH"
3. Click "Install Now" or "Customize installation"
4. Wait for installation to complete

STEP 3: Verify Installation
---------------------------
1. Open Command Prompt (search for "cmd" in Start menu)
2. Type: python --version
3. You should see: Python 3.x.x
4. Type: python -c "import PyQt6, serial, can, obd"
5. If you see errors, continue to Step 4

STEP 4: Install Required Packages
---------------------------------
1. Open Command Prompt as Administrator
2. Type: python -m pip install PyQt6 pyserial python-can obd
3. Wait for installation to complete

STEP 5: Test AutoDiag Pro
-------------------------
1. Run AutoDiag_Launcher.bat from the installation directory
2. The launcher should start successfully

TROUBLESHOOTING
---------------
- If Python is not found, restart your computer
- If packages fail to install, try: pip install --upgrade pip
- For permission errors, run Command Prompt as Administrator
