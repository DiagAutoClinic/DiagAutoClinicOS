DACOS Authority Tiers

DACOS does not sell "features". DACOS grants system authority over vehicles. Authority without structure becomes damage.

Tier 1: Free - Observer
Negligible Risk

Security Authority: Generic OBD-II only, Read-only CAN frames, No ECU state mutation possible

Capabilities:

  • VIN decoding
  • Read / clear emission-related DTCs
  • Live sensor data
  • Snapshot logging

Price: R0 / month

For: Verification, Education, Trust building

Tier 2: Basic - Maintenance Operator
Low Risk

Security Authority: Limited CAN write access, No security codes, No gateway override

Capabilities:

  • Service resets
  • Maintenance-related functions
  • Full-module DTC read/clear

Price: R249 / month

For: DIY users, Roadside technicians, Entry-level workshops

Tier 3: Intermediate - System Influencer
Moderate Risk

Security Authority: Controlled write access, Learning and adaptation functions

Capabilities:

  • Throttle learning
  • Steering angle calibration
  • Battery registration
  • Actuator tests

Price: R599 / month

Requires Acknowledgement

For: Independent workshops, Diagnostic specialists

Tier 4: Professional - Vehicle Authority
High Risk

Security Authority: Dealer-level CAN access, Gateway-aware operations, Multi-bus control

Capabilities:

  • System calibrations
  • Bidirectional ECU control
  • Secure gateway workflows
  • Advanced diagnostics

Price: R1,299 / month

Requires Per-Session Acknowledgement

For: Professional diagnostic centers, Specialist repair shops

Tier 5: Advanced - System Architect
Extreme Risk

Security Authority: Raw CAN frame injection, ECU coding and configuration, Full security-level access

Capabilities:

  • Variant coding
  • ECU parameter modification
  • Custom CAN definitions
  • Raw frame injection and replay

Price: R2,499 / month

Requires Explicit & Recurrent Acknowledgement

For: ECU engineers, Security researchers, Advanced coders
